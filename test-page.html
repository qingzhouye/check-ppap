🌐 Chrome 扩展测试提示

当前配置: 使用 Manifest V3 + service_worker（Chrome 推荐格式）

Chrome 加载步骤:

  1. 地址栏输入 chrome://extensions/
  2. 打开右上角 "开发者模式"
  3. 点击 "加载已解压的扩展程序"
  4. 选择 consistency-checker-extension 文件夹

调试方法:

  • 查看后台脚本日志：扩展卡片 → 点击 "service worker" 链接
  • 查看 popup 日志：右键扩展图标 → "检查弹出内容"
  • 查看内容脚本日志：目标页面按 F12 → 控制台
