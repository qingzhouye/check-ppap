# 一致性确认助手 - 使用说明

## 一、插件简介

**一致性确认助手**是一款专为 SGMW 供应商质量工程师(SQE)设计的浏览器插件，用于自动化"一致性确认"任务的校验和审核工作。

### 主要功能
- **自动提取任务列表**：从页面快速获取所有待处理任务
- **智能一致性校验**：自动比对零件信息、CCC标识、型号规格等
- **AI图像识别**：自动识别CCC证书和型号标识图片内容
- **批量校验**：自动逐个打开任务详情页进行校验
- **批量审核**：一键自动填写监测组审核意见并提交

---

## 二、安装方法

### 方式一：开发者模式加载（推荐）

1. 打开 Chrome/Edge 浏览器，进入扩展管理页面：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`

2. 开启右上角的**"开发者模式"**开关

3. 点击**"加载已解压的扩展程序"**

4. 选择 `consistency-checker-extension` 文件夹

5. 安装成功后，浏览器工具栏会出现插件图标

---

## 三、界面说明

### 1. 插件主面板

点击浏览器工具栏的插件图标打开主面板，包含以下区域：

| 区域 | 说明 |
|------|------|
| **状态栏** | 显示已导入Excel数量和AI识别服务状态 |
| **操作功能** | 提取任务、一键校验、批量校验、自动审核 |
| **Excel管理** | 导入/管理关键件清单Excel文件 |
| **API Key** | 大模型AI识别服务的密钥配置 |
| **批量校验结果** | 显示批量校验的详细结果和状态 |
| **操作日志** | 记录所有操作和校验结果 |

### 2. 页面悬浮按钮

在 SGMW 一致性确认页面右下角会出现蓝色悬浮按钮：
- **一致性助手**：点击展开快捷菜单
- 可自由拖动到任意位置

## 四、使用步骤

### 第一步：导入关键件清单

1. 点击插件面板中的 **"新增Excel文件"** 按钮
2. 选择 E262C 关键件清单 Excel 文件（支持多选）
3. 等待导入完成，状态栏会显示导入数量

> **注意**：Excel 文件格式需符合标准模板，包含零件名称、CCC标识、型号规格、生产企业等字段

### 第二步：提取任务列表

1. 进入 SGMW 一致性确认任务列表页面
2. 点击插件面板的 **"提取当前任务列表"** 按钮
3. 或在页面点击悬浮按钮 → **"提取任务列表"**
4. 日志区会显示提取到的任务数量和详情

### 第三步：单个任务校验

#### 方式一：当前详情页校验
1. 在任务列表中点击任意任务，打开详情弹窗
2. 点击插件面板的 **"一键校验当前详情页"** 按钮
3. 或在页面点击悬浮按钮 → **"一键校验"**
4. 等待 AI 识别完成，页面右上角会显示校验结果面板

#### 方式二：批量校验
1. 确保已在任务列表页面
2. 点击插件面板的 **"批量校验任务清单"** 按钮
3. 确认开始批量校验
4. 插件会自动逐个打开任务详情页进行校验
5. 校验完成后在 **"批量校验结果"** 区域查看结果

### 第四步：查看校验结果

校验结果面板会显示以下内容：

| 校验项 | 说明 |
|--------|------|
| 基本信息 | 零件名称、零件号、车型 |
| 生产企业名称一致性 | 比对供应商与生产企业名称 |
| 是否CCC件 | 比对页面与Excel的CCC标识 |
| 型号信息 | 比对页面与Excel的型号规格 |
| CCC标识(AI识别) | AI识别实物图片中的CCC标志 |
| 型号标识(AI识别) | AI识别实物图片中的型号信息 |

**结果标识**：
- ✓ 绿色 - 校验通过
- ✗ 红色 - 校验不通过
- ! 黄色 - 需要人工确认

### 第五步：人工审核

对于标记为 **"需人工"** 的任务：

1. 在批量校验结果列表中找到该任务
2. 点击任务行展开详情，查看具体校验结果
3. 根据实际情况点击：
   - **确认通过** - 标记为人工审核通过
   - **确认不通过** - 标记为人工审核不通过（需填写备注）
4. 也可点击任务行打开详情弹窗进行审核

### 第六步：批量审核提交

1. 批量校验完成后，点击 **"一键审核通过并提交"** 按钮
2. 确认开始批量审核
3. 插件会自动：
   - 打开校验通过的任务详情页
   - 自动选择"关键件清单一致=是"
   - 自动选择"确认结果=同意"
   - 自动点击提交按钮
4. 审核完成后会显示处理结果

## 五、窗口拖动功能

所有插件窗口都支持自由拖动：

| 窗口 | 拖动方式 |
|------|----------|
| 插件主面板 | 按住顶部标题栏拖动 |
| 悬浮按钮 | 直接按住拖动 |
| 悬浮菜单 | 按住菜单标题栏拖动 |
| 校验结果面板 | 按住蓝色标题栏拖动 |

**提示**：
- 拖动时会自动切换为绝对定位
- 窗口位置会被限制在浏览器视窗内
- 双击插件主面板标题栏可重置位置

---

## 六、API Key 配置

插件内置了默认的 AI 识别服务密钥，如需更换：

1. 在 **"大模型API Key"** 区域输入新的密钥
2. 点击 **"保存"** 按钮
3. 点击 **"恢复默认"** 可恢复内置密钥

---

## 七、数据持久化

插件会自动保存以下数据：

- **导入的Excel文件列表** - 关闭浏览器后仍然保留
- **批量校验结果** - 刷新页面后可恢复查看
- **操作日志** - 最近50条记录会保留

**清空数据**：
- 点击批量结果区域的 **"清空结果"** 可清空所有校验结果和日志
- 或在日志区域点击 **"清空"** 仅清空日志

**导出日志**：
- 点击日志区域的 **"导出日志"** 可将操作记录保存为文本文件

## 八、常见问题

### Q1: 插件无法正常工作？
- 确保在正确的 SGMW 一致性确认页面使用
- 检查浏览器是否已启用开发者模式
- 尝试刷新页面后重新打开插件

### Q2: AI识别失败？
- 检查网络连接是否正常
- 确认 API Key 是否有效
- 查看日志区的具体错误信息

### Q3: 批量校验卡住？
- 可能是页面弹窗未正确关闭，手动关闭后继续
- 检查网络连接是否稳定
- 如长时间无响应，可刷新页面重新开始

### Q4: 校验结果不准确？
- 确保已导入正确的关键件清单Excel
- AI识别结果仅供参考，重要决策请人工复核
- 对于模糊图片，建议下载后人工查看

---

## 九、注意事项

1. **使用前请仔细核对** AI 识别结果，重要决策需人工复核
2. **批量审核功能** 会自动提交表单，请确保校验结果正确后再使用
3. **PDF附件** 无法自动识别，需要人工下载查看
4. **建议先进行小批量测试**，确认无误后再进行大批量操作
5. **定期导出日志**，以便追溯操作记录
